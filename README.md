# Telegram Bot

A Telegram bot built with aiogram and Python.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Get your bot token from [@BotFather](https://t.me/BotFather) on Telegram

3. Update `main.py` and replace `YOUR_BOT_TOKEN` with your actual bot token

4. Run the bot:
```bash
python main.py
```

## Structure

- `main.py` - Entry point of the application
- `app/handlers/` - Command and message handlers
- `app/services/` - Business logic services
- `app/utils/` - Utility functions
- `app/keyboards/` - Inline and reply keyboards
